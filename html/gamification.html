Regole della Gamification

Missioni Giornaliere

  • “Riscaldamento quotidiano” - Registra un allenamento oggi. → +10 XP
  • “Costanza prima di tutto” - Registra due allenamenti in un giorno. → +20 XP
  • “Equilibrio perfetto” - Allenati almeno 1 ora. → +20 XP
  • “Spingi oltre” - Supera il numero di ripetizioni del giorno precedente. → +15 XP
  • “Cardio Power” - Completa almeno 20 minuti di cardio. → +15 XP
  • “Sollevatore del giorno” - Fai almeno 3 esercizi con i pesi. → +15 XP
  • “Forza esplosiva” - Aumenta di 1kg il peso rispetto all’ultima sessione. → +20 XP
  • “Resistenza d'acciaio” - Allenati almeno 120 minuti. → +30 XP

Missioni Settimanali

  • “Settimana di fuoco” - Completa almeno 3 allenamenti. → +50 XP
  • “Costanza assoluta” - Allenati per 7 giorni consecutivi. → +100 XP
  • “Miglioramento continuo” - Aumenta i carichi in 3 esercizi. → +75 XP
  • “Resistenza Pro” - Accumula 5 ore totali di allenamento. → +75 XP
  • “Brucia tutto” - Consuma 3.000 kcal in allenamento. → +75 XP
  • “Riconosci il tuo progresso” - Aggiorna peso e misure. → +50 XP
  • “Sfida muscolare” - Aggiungi un nuovo esercizio. → +50 XP

Missioni Mensili

  • “Mese da atleta” - Completa 12 sessioni nel mese. → +150 XP
  • “Massimo impegno” - 20 ore di allenamento. → +200 XP e badge “Atleta Dedicato”
  • “Forza oltre i limiti” - Aumenta carico in 5 esercizi. → +150 XP
  • “Sfida della resistenza” - Corri/pedala 50 km. → +200 XP e badge “Endurance King”
  • “Scolpisci il tuo corpo” - 4 sessioni full body. → +150 XP e cornice profilo
  • “Maratona di allenamento” - 20 allenamenti nel mese. → +250 XP e boost XP
  • “Crescita muscolare” - +2% massa magra. → +250 XP e badge “Muscle Builder”
  • “Precisione e controllo” - Registra tutto con precisione. → +150 XP
